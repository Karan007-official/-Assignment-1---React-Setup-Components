# Assignment 1 - React Setup & Components

## 📌 Project Overview
This project is a basic React application built using **Vite**. It includes fundamental UI components like Navbar, Sidebar, Footer, and a simple Home Page layout.

---

## ⚙️ Tech Stack
- React (Vite)
- JavaScript (ES6+)
- HTML5
- CSS3

---

## 🚀 Setup Instructions

### 1. Create React Project using Vite
```bash
npm create vite@latest my-react-app
cd my-react-app
npm install
npm run dev
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

---

## 🧩 Components Created

### 1. Navbar Component
- Contains logo/title
- Navigation links (Home, About, Contact)

### 2. Sidebar Component
- Side navigation menu
- Links or menu items for layout structure

### 3. Footer Component
- Copyright text
- Basic footer information

### 4. Home Page UI
- Displays main content area
- Combines Navbar, Sidebar, and Footer
